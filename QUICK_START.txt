📊 FÓRMULAS IMPLEMENTADAS

Óptica:
  FOV = (SensorDim / Focal) * (WD + Focal)
  Magnification = SensorWidth / FOV

Motion Blur:
  Blur = (Velocidad / TamañoPíxel) * Exposición / 1000

Profundidad de Campo:
  H = (f² / (N * c)) + f
  Near = (H * s) / (H + (s - f))
  Far = (H * s) / (H - (s - f))

Legibilidad:
  PixelsPerModule = ModuleSize / mmPerPixel
  Verdict: readable si ≥ threshold * 2

❓ TROUBLESHOOTING

Error: "Cannot find module..."
→ npm install

Error: "NEXT_PUBLIC_SUPABASE_URL is undefined"
→ Verifica .env.local existe y contiene variables

Puerto 3000 ocupado:
→ npm run dev -- -p 3001

Cálculos no funcionan:
→ F12 → Console tab → mira errores
→ Verifica valores de entrada sean positivos


💡 NOTAS IMPORTANTES

⚠️  Fórmulas usan aproximación paraxial (WD >> Focal)
⚠️  Motion blur es teórico, no simula óptica real
⚠️  DOF usa distancia hiperfocal clásica
⚠️  Códigos: AIM/ISO, depende de contraste físico
